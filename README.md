# bdubs.vim

> quickly delete and wipeout buffers

- `:BD[N]` to delete all but the `N` most recently used buffers

- `:BW[N]` to wipe out all but the `N` most recently used buffers

See [`doc/bdubs.txt`](doc/bdubs.txt) for more details.
